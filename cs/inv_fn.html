Inverse Functions

A function $f:X \to Y$ maps elements in the set $X$ to elements in the set $Y$ so that each $x \in X$ gets sent to at most one element in $Y$. A function $f$ is 1-to-1 if for every possible element $y$ in the set $Y$ there is only one $x$ that maps to it (ie. $f(x)=y$).

Give an example of a function that is 1-to-1. Give an example of a function that is not 1-to-1. Explain.
$f(x)=2x$ is 1-to-1. We know this since the line is always increasing at rate of 2 units $y$ per unit $x$, hence each $y$ can be associated to one and only one $x$. $h(x)=x^2$ is not 1-to-1 since both 2 and -2 are sent to 4.

If we take a moment to make a table of values for the two exponentials $f(x)=2^x$ and $g(x)=10^x$ we can see that both of these functions are always increasing, and most likely are 1-to-1.
