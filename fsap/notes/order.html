Order of Operations

We have to agree on how to evaluate expressions

We need to agree on how to read a series of numbers and operations on those numbers so that we can all agree as to what it evaluates to. In the absence of established rule as to the order of operations, the expression $2\cdot 3+4$ could evaluate to 14 or 10 depending on whether one does addition or multiplication first. This would make reading someone else's mathematical work very difficult if you had to adapt to the author's convention each time you read a different author. It is important to realize that PEMDAS is just a convention that we all agree upon, similar to driving on the right side of the road. It is unlike other mneumonic devices in that it is an arbritrary convention. Compare this to All Student Take Calculus as a way to remember the respective signs of trigonometric functions in each quadrant of the coordinate plane. If you understand the trig function, you don't need the mneumonic. It is not convention but definition and reasoning that underlie the mneumonic.

Some examples of common errors.
  • $4(2)^3\neq 8^3$
  • $-3^2 \neq 9$
  • $(3+5)^2 \neq 3^2+5^2$
  • $\sqrt{3^2+5^2} \neq 3 +5$

It is necessary to be able to read mathematical expressions encountered in various contexts, but there are some practices in writing expressions that I think can clarify certain vague cases in using PEMDAS.

Operations with Functions

You can create new functions from old functions using the arithmetic operations. For example, let $f(x)=2x$ and $g(x)=x^2-x$. Then we can define a new function $h$ using $f$, $g$, and addition, subtraction, multiplication or division.

  • $h(x)=(f+g)(x)=f(x)+g(x)$
  • $h(x)=(f\cdot g)(x)=f(x)\cdot g(x)$
  • $h(x)=(f/g)(x)= \dfrac{f(x)}{g(x)}$ where $g(x) \neq 0$.

We can also define a new function using the operation called composition.

Composition of Functions

Given functions $f:X\to Y$ and $g:Z\to X$, the function $h(x)=(f \circ g)(x):Z \to Y$ is the composition of $f$ and $g$ it sends $z \to g(z)=x \to f(g(z))=f(x)=y$.

Let $f(x)=x^2+1$ and $g(x)=3x-4$. Then the new function $f$ composed with $g$ can be written as either $(f\circ g)(x)$ or $f(g(x))$. And since $f$ and $g$ are nice polynomials you can give the explicit rule for $f$ composed with $g$ as follows:

$\begin{array}{r l} f(g(x))& =f(3x-4)\\ & = (3x-4)^2+1\\ & = 9x^2-24x+16+1\\ & =9x^2-24x+17 \end{array}$

We are not restricted to polynomials. Let $f(x)=|2x-1|$ and $g(x)=3\sin (\pi x)$. Then $(g \circ f)(x)$ can be found as follows:

$\begin{array}{r l} (g \circ f)(x) &= g(f(x))\\ &= g(|2x-1|)\\ &= 3\sin(\pi|2x-1|) \end{array}$

A good way to build intution as to what these operations with functions create is by looking at the graphs of some of the outcomes. For each of the following pairs of functions first create the new function algebraically as indicated by the operation between them, and then try to sketch what this new function looks like. Lastly actually check your sketch in desmos by following the link.

  1. Let $f(x)=2x$ and $g(x)=-2x$. Form $(f+g)(x)$, $(f-g)(x)$, $(fg)(x)$, and $(f/g)(x)$. Check your sketches.
  2. This one's a little harder. Let $f(x)=x^2$ and $g(x)=2\sin (3x)$. Form $(f+g)(x)$, $(f-g)(x)$, $(fg)(x)$, and $(g/f)(x)$. Check your sketch.
  3. Let $f(x)=|x|$ and $g(x)=\sin x$. Form $f(g(x))$. Check the sketch yourself using Desmos.
  4. Try to figure out why the new functions look the way they look.

Usefulness of Function Notation

There are some good reasons to adopt this notation. Previously we would think of $y=3x$ and $y=x^2+\sqrt{x}$ as adequate notation for functions. Changing notation to $f(x)=3x$ and $g(x)=x^2+\sqrt{x}$, we can now refer to each function in terms of the label of their outputs. For example we can say $f$ is linear and $g$ is a square root function without any confusion. This is becomes more helpful as an organizational tool as the number of equations we work with increases. This notation also makes the operation of composing function more clear, as the inputs and outputs for each equation are clearly labeled so that linking two functions together is suggested by the notation $f(g(x))$ where $x \to g \to f$.
