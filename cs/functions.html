Functions

Maps and Functions

A mapping is an assignment of elements of one set to elements in another set. Typically it is notated as $f:X \to Y$, where $f$ is the name of the map, $X$ is the set of inputs and $Y$ is a set that contains the outputs. A function is a mapping where each element from the set of inputs is assigned to one and only one element from the set of outputs.

Let's take as our first examples two maps $f$ and $g$. The map $f$ sends integers to their doubles. First we can express that $f$ is a map from the set of integers to the set of integers with $f: \mathbb{Z} \to \mathbb{Z}$. We can sometimes also give a rule to generate the output from a given input $x$ with the notation $f: x\mapsto 2x$. Lastly we can talk about how specific elements are mapped by writing $1 \mapsto 2$ or $13 \mapsto 26$.

The mapping $g$ will send any number to its square. Following the notation convention just established we have that $g:\mathbb{R}\to \mathbb{R}$ and $g:x \mapsto x^2$.

Make a chart of the inputs and and outputs of $g$ for inputs $= \{-2,-1,0,1,2,3, \pi \}$.
Explain why both $f$ and $g$ qualify as functions according to the definition given above.

As we have already noted $f$ and $g$ are both functions. Notice that we defined $f$ as taking only integers inputs, whereas $g$ can take any real number as an input. Also, the outputs of $f$ are the multiples of 2, subset of the integers, while the outputs of $g$ form the nonnegative real numbers. We can use the terms domain and image(or range) to describe a function's set of inputs and set of outputs, respectively.

Domain and Range

The domain of a function $f$ is the set of inputs for which the function is defined. The range or image of a function $f$ is the set of outputs attained by $f$ for a given domain.

Finally we should note that going forward we will use function notation to denote functions instead of the mapping notation introduced above. The two functions already introduced will be notated as $f(x)=2x$ and $g(x)=x^2$. It is read as "f of x equals 2x" and "g of x equals x squared". It may be tempting to read the $f(x)$ as $f$ times $x$, but the difference is usually clear from context.

Operations with Functions

You can create new functions from old functions using the arithmetic operations. For example, let $f(x)=2x$ and $g(x)=x^2-x$. Then we can define a new function $h$ using $f$, $g$, and addition, subtraction, multiplication or division.

  • $h(x)=(f+g)(x)=f(x)+g(x)$
  • $h(x)=(f\cdot g)(x)=f(x)\cdot g(x)$
  • $h(x)=(f/g)(x)= \dfrac{f(x)}{g(x)}$ where $g(x) \neq 0$.

We can also define a new function using the operation called composition.

Composition of Functions

Given functions $f:X\to Y$ and $g:Z\to X$, the function $h(x)=(f \circ g)(x):Z \to Y$ is the composition of $f$ and $g$ it sends $z \to g(z)=x \to f(g(z))=f(x)=y$.

Let $f(x)=x^2+1$ and $g(x)=3x-4$. Then the new function $f$ composed with $g$ can be written as either $(f\circ g)(x)$ or $f(g(x))$. And since $f$ and $g$ are nice polynomials you can give the explicit rule for $f$ composed with $g$ as follows:

$\begin{array}{r l} f(g(x))& =f(3x-4)\\ & = (3x-4)^2+1\\ & = 9x^2-24x+16+1\\ & =9x^2-24x+17 \end{array}$

We are not restricted to polynomials. Let $f(x)=|2x-1|$ and $g(x)=3\sin (\pi x)$. Then $(g \circ f)(x)$ can be found as follows:

$\begin{array}{r l} (g \circ f)(x) &= g(f(x))\\ &= g(|2x-1|)\\ &= 3\sin(\pi|2x-1|) \end{array}$

Usefulness of Function Notation

There are some good reasons to adopt this notation. Previously we would think of $y=3x$ and $y=x^2+\sqrt{x}$ as adequate notation for functions. Changing notation to $f(x)=3x$ and $g(x)=x^2+\sqrt{x}$, we can now refer to each function in terms of the label of their outputs. For example we can say $f$ is linear and $g$ is a square root function without any confusion. This is becomes more helpful as an organizational tool as the number of equations we work with increases. This notation also makes the operation of composing function more clear, as the inputs and outputs for each equation are clearly labeled so that linking two functions together is suggested by the notation $f(g(x))$ where $x \to g \to f$.
